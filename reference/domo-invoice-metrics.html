⚠️ Metrics Requiring Separate SQL Tasks

The following metrics shown in this dashboard are not covered by the current invoice SQL task and require additional queries or dataset joins before they can be populated in Domo:

Metric Why It Needs a Separate Task Status
Early Pay Discount Capture Requires JOIN with payment/discount tables — not in current invoice dataset ⏳ Separate task needed
